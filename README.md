# Columbidae Client

LAVENANT Jordan & ROUSSEL Marc - APP3 IIM

## Description

Client du projet **Columbidae**, qui consomme l'API et affiche les données (réseau social).

## Pre-requisites

- Node.js (20+)

## Installation

1. Cloner le dépôt GitHub

```bash
git clone https://github.com/jordanlavenant/columbidae-client.git
```

2. Accéder au répertoire du projet

```bash
cd columbidae-client
```

3. Installer les dépendances

```bash
npm install
```

4. Configurer les variables d'environnement
   Créer un fichier `.env` à la racine du projet et ajouter les variables nécessaires (voir `.env.example` pour référence).

```bash
cp .env.example .env
```

```
VITE_API_ENDPOINT=<your api endpoint>
```

## Démarrer le serveur de développement

```bash
npm run dev
```
